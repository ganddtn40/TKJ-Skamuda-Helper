# 🛠️ TKJ Skamuda Helper 📱

**Buku Saku Digital Wajib Siswa SMK Jurusan Teknik Komputer & Jaringan (TKJ)**

[![Design](https://img.shields.io/badge/Style-SK--Modernist-white?style=flat-square&color=000000)](https://github.com/ganddtn40/TKJ-Skamuda-Helper)

Aplikasi **TKJ Skamuda Helper** adalah pendamping digital yang dirancang khusus untuk mempermudah praktikum dan teori bagi siswa TKJ. Dengan desain yang futuristik dan navigasi yang intuitif, aplikasi ini membantu siswa menguasai konsep jaringan, perangkat keras, dan IoT dalam satu genggaman.

---

## ✨ Fitur Utama (Core Features)

### 🌐 Networking (Jaringan Komputer)

- **Advanced IP Calculator**: Menghitung Subnetting (NetID, Broadcast, Subnet Mask) secara instan.
- **Subnetting Practice**: Latihan soal subnetting acak untuk persiapan ujian teori kejuruan.
- **MikroTik CLI Cheat Sheet**: Daftar command dasar RouterOS, default IP login, dan checklist konfigurasi gateway.
- **Troubleshooting OSI Flowchart**: Panduan interaktif mencari sumber masalah jaringan dari Layer 1 sampai Layer 7.
- **LAN Cable Guide**: Panduan visual urutan kabel UTP (Straight-through & Crossover).

### ⚙️ Hardware (Perangkat Keras)

- **Resistor Calculator**: Membaca nilai resistor 4 & 5 gelang secara akurat.
- **Sensor Dictionary**: Wiring dan deskripsi pinout sensor umum (DHT11, HC-SR04, Relay, dll.).
- **BIOS Beep Codes**: Referensi bunyi beep BIOS (AMI & Award) untuk diagnosa kerusakan PC.

### 🤖 IoT & Coding

- **Board Pinouts**: Peta visual pinout NodeMCU ESP8266 dan ESP32.
- **IoT Snippets**: Template kode Wi-Fi, MQTT, dan HTTP siap pakai untuk Arduino IDE.

## 🎨 Design Philosophy: SK-Modernist

Aplikasi ini mengadopsi estetika **SK-Modernist** yang premium:

- **Minimalist Monochrome**: Palet warna Putih (`#FFFFFF`), Abu-abu (`#F2F2F2`), dan Hitam (`#000000`).
- **Liquid Glass Effect**: Implementasi _glassmorphism_ dengan border halus dan transparansi tinggi.
- **Fluid Motion**: Menggunakan `react-native-reanimated` (v4.x) untuk animasi _spring_ yang kenyal dan transisi layar yang halus.

## 🛠️ Menjalankan Secara Lokal

1. **Clone repository**:
   ```bash
   git clone https://github.com/ganddtn40/TKJ-Skamuda-Helper.git
   ```
2. **Install dependensi**:
   ```bash
   npm install
   ```
3. **Mulai aplikasi**:
